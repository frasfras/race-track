# Product Overview

Driver Insights is a React dashboard application for analyzing racing driver performance data. The app provides comprehensive analytics for racing drivers by visualizing telemetry data, performance metrics, and track behavior patterns.

## Core Features

- **Driver Selection**: Filter and analyze data by specific driver IDs
- **Racing Mindset Profile**: Radar charts displaying 6 key performance metrics (Aggression, Consistency, Composure, Smoothness, Adaptability, Strategy)
- **Performance Analytics**: Horizontal bar charts with color-coded performance breakdowns
- **Track Visualization**: GPS-based scatter plots showing throttle data and corner analysis
- **Summary Dashboard**: Driver profiles with metric descriptions and numerical scores

## Target Users

Racing teams, drivers, and analysts who need to understand driver performance patterns and track behavior for training and strategy optimization.
